  # CraftyBay

## A simple E-commerce app  UI like Daraz to implement pixel to flutter UI with full api implement 
# Features
- **Splash Screen**
- **Home Screen**
- **Product Details Screen**
- **Login Screen**
- **Loading Screen**
- **Cart Screen**
- **OTP Verify Screen**
- **Popular Product Screen**
- **Special Product Screen**
- **New Product Screen**
- **Reviews Screen**
- **Create Review Screen**
- **Wish List**
- **Categories List**
